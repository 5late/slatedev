PepeBot

Languages Used: Python || System Admin (Bash Scripting)

Hours: ~120

What I did

For PepeBot I wrote code in Python to use the Discord API and make a simple bot. PepeBot has simple commands to let me set reminders, send links, report statistics, and much more.

It also connects with the VALORANT API to server stats about players, which is what many other people use it for. It's partly open source at this GitHub repo!

I also used my web server to host the bot, and to collect error logs if the bot failed using some bash commands.

What I learned

Python (learned how to use the language)

How to use the Discord API through the discord.py library.

How to collect error logs from commands using bash commands and text files.
