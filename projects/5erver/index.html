What I did

For 5erver, I started by making a simple yet beautiful landing page. I wanted it to be simple, easy to use, and clutter-free. I settled on a darker theme, and used HTML & CSS to code it.

I then wrote the backend logic in Go, to upload files. The Golang backend runs in the background and downloads files onto the disk.

Lastly, I wrote the JavaScript to connect everything. It makes requests to the Go backend, which returns html pages to display while it does the file transaction. It's all open source at this GitHub repo!

What I learned

Go net/http package

CSS Styling for a pretty landing page.

How to upload files to a backend, and then make them accessible from the front end.
