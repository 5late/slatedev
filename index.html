What I did

For the Newb Config set of scripts, I learned some fairly basic shell scripting to create a set of scripts that I think would make it easier for a newer use to Linux to use the terminal.

The purpose was to create scripts that would not interfere, with the user, but rather work in the background. It is all open source at this GitHub repo!

I made the scripts as modular as possible, so users could decide which features they installed, and which they did not. I also documented the entire project in depth through the README and comments in the main install script.

To install the script without having to clone the repo, you can download/wget from this link.

To see the script in plain text before you install it and run it, see this link.

I suggest reading #installation in the README of the github repo which you can find here.
